# AGENTS.md

이 저장소에서 작업하는 AI 코딩 에이전트 공통 작업 규칙입니다.

## Mandatory TODO Workflow

1. 작업 시작 전에 루트 `TODO.md`를 읽고 현재 남은 항목을 파악한다.
2. 작업 중에는 세션 TODO 목록을 지속적으로 업데이트한다.
3. 작업이 끝나면 루트 `TODO.md`를 반드시 최신 상태로 갱신한다.
4. 최종 보고에는 다음 두 가지를 반드시 포함한다.
   - 이번 작업으로 완료된 TODO 항목
   - 아직 남아 있는 TODO(remain tasks) 리마인드
5. `TODO.md`의 `### Remaining`은 반드시 기능 단위 섹션(`#### <SECTION>`)으로 유지한다.

## Completion Rule

- "완료"라고 말하기 전에 `TODO.md`의 관련 항목 상태가 실제 코드/문서 상태와 일치해야 한다.
- 남은 작업이 하나라도 있으면 최종 응답에서 명시적으로 알려준다.
- remain tasks를 리마인드할 때는 항목뿐 아니라 해당 `#### <SECTION>` 이름도 함께 안내한다.
